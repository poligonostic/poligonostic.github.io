About Poligonostic Weather

Poligonostic Weather is a fully custom-built weather visualization system designed to be fast, beautiful, and completely free. It uses open meteorological data, custom animations, and a unique location picker to create an interactive weather experience.

Data Sources

  • Open‑Meteo API — Forecast, hourly data, daily data, weather codes
  • OpenStreetMap — Map tiles for the location picker
  • Leaflet.js — Mapping engine
  • Custom Poligonostic Radar Plugin — Radar visualization

Features

  • Interactive map-based location picker
  • Realistic weather animations (sun, clouds, rain, snow, lightning)
  • 7-day forecast
  • Hourly conditions
  • Metric/Imperial toggle
  • Favorites system
  • City + State formatting
  • Fast loading with zero API keys
