Poligonostic Leaflet Radar Plugin

The Poligonostic Leaflet Radar Plugin is a Leaflet extension for displaying animated LibreWXR radar, satellite imagery, and weather alerts. It is officially listed in the Leaflet Plugin Directory under overlay-animations.

Installation:

NPM Install:

npm install poligonostic-leaflet-radar

CDN / Script Tag:

<script src="poligonostic-leaflet-radar.js"></script>

Basic Usage

const radar = L.poligonosticRadar({
  opacity: 0.8,
  animate: true,
  smooth: true,
  satellite: true,
  alerts: true,
  color: "classic",
  updateInterval: 60000
});

radar.addTo(map);

Options

  • opacity — radar transparency (0–1)
  • animate — enable radar animation
  • smooth — fade transitions between frames
  • satellite — toggle satellite imagery
  • alerts — toggle weather alerts overlay
  • color — radar color palette
  • updateInterval — auto-refresh interval in ms

Radar Color Palettes

Available color modes:

  • "classic" — standard radar colors
  • "blue" — blue-scale precipitation
  • "neon" — bright neon radar
  • "grayscale" — black & white radar

Example:

const radar = L.poligonosticRadar({
  color: "neon"
});

Satellite & Alerts Toggles

Turn satellite on/off:

radar.setSatellite(true);   // enable
radar.setSatellite(false);  // disable

Turn alerts on/off:

radar.setAlerts(true);   // enable
radar.setAlerts(false);  // disable

API Reference

  • setOpacity(value)
  • setColor(mode)
  • setSatellite(bool)
  • setAlerts(bool)
  • refresh() — manually reload frames
  • remove() — remove radar layer
